# React Weather App 🌤️

A responsive weather app that shows real-time weather and a 5-day forecast for any city in the world.

## Live site

[View live](https://enchanting-blancmange-072e34.netlify.app/)

## Built with

- React
- Tailwind CSS for styling
- Vite
- OpenWeatherMap API

## Features

- Search any city for live weather data
- Current temperature, humidity and wind speed
- 5-day forecast showing temp, wind speed and description
- Loading state while data is fetching
- Error message when city is not found
- Enter key support for search

## React concepts used

- Components and props
- useState for weather, forecast, loading and error state
- Conditional rendering
- Data flows from App to child components
- Controlled inputs

## JS concepts used

- async/await with try/catch/finally
- Promise.all for two API response at once
- Array destructuring on API response data
- map() and filter() Array methods
- Template literals for dynamic URLs
- Optional chaining on nested API objects
